# Ideation Prompt

## Context
프로젝트 브리프를 기반으로 다양한 아이디어를 발산합니다.

## Input
{{PROJECT_BRIEF}}

## Task
위 프로젝트 개요를 바탕으로 다음을 수행해주세요:

1. **핵심 기능 아이디어** (최소 10개)
   - 각 아이디어에 대한 간단한 설명
   - 예상 구현 난이도 (상/중/하)
   - 사용자 가치

2. **기술적 접근법**
   - 가능한 기술 스택 옵션
   - 각 옵션의 장단점

3. **차별화 포인트**
   - 경쟁 제품 대비 차별화 요소
   - 혁신적인 기능 제안

4. **MVP 범위 제안**
   - 최소 기능 제품에 포함될 기능
   - 우선순위 기준

## Output Format
마크다운 형식으로 구조화하여 출력해주세요.
